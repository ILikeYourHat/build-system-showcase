# Maven

Official page: [https://maven.apache.org](https://maven.apache.org)

Repository: [https://github.com/apache/maven](https://github.com/apache/maven)

---

## Scripts

Building app:
```bash
./mvnw compile
```

Running app:
```bash
./mvnw compile exec:java
```

Testing app:
```bash
./mvnw test
```
